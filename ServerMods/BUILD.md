# Инструкция по сборке и установке Launcher API Mod

## Требования

- Java 21
- Gradle (или используйте gradlew wrapper)

## Сборка

1. Откройте терминал в папке `ServerMods`
2. Выполните команду:

**Windows:**
```bash
gradlew.bat build
```

**Linux/Mac:**
```bash
./gradlew build
```

3. Готовый мод будет в папке `build/libs/launcherapi-1.0.0.jar`

## Установка на сервер

1. Скопируйте файл `launcherapi-1.0.0.jar` в папку `mods/` вашего сервера
2. Убедитесь, что у вас установлен Fabric Loader 0.16.9+ и Fabric API
3. Перезапустите сервер
4. В логах должно появиться сообщение: `HTTP API сервер запущен на порту 30082`

## Проверка работы

После запуска сервера проверьте доступность API:

```bash
curl http://213.171.18.211:30082/api/players
```

Или откройте в браузере: `http://213.171.18.211:30082/api/players`

## Порты

По умолчанию API работает на порту **30082**. 

Если нужно изменить порт, отредактируйте файл:
`ServerMods/src/main/java/com/launcher/api/HttpApiServer.java`

Найдите строку:
```java
httpServer = new HttpApiServer(30082, server);
```

И измените `30082` на нужный порт.

## Устранение проблем

### API не отвечает
- Проверьте, что мод загружен (смотрите логи сервера)
- Убедитесь, что порт 30082 не занят другим приложением
- Проверьте файрвол - порт должен быть открыт

### Ошибка при сборке
- Убедитесь, что Java 21 установлена
- Проверьте версию Fabric Loader и API в `gradle.properties`
