# Быстрый старт

## 1. Сборка мода

```bash
# Windows
gradlew.bat build

# Linux/Mac
./gradlew build
```

Готовый мод: `build/libs/angella-1.0.0.jar`

## 2. Установка на сервер

1. Скопируйте `angella-1.0.0.jar` в папку `mods` вашего сервера
2. Убедитесь, что установлены:
   - Fabric Loader 0.16.9+
   - Fabric API
3. Запустите сервер

## 3. Настройка Discord бота

1. После первого запуска откройте `config/angella.json`
2. Укажите:
   - `botToken` - токен вашего Discord бота
   - `channelId` - ID канала Discord (включите режим разработчика в Discord)
3. Перезапустите сервер

## 4. Настройка SkinRestorer (опционально)

Если у вас установлен SkinRestorer:

1. В `config/angella.json` установите `useSkinRestorer: true`
2. Если SkinRestorer предоставляет HTTP API, укажите `skinRestorerApiUrl`
3. Или оставьте пустым - мод автоматически попытается найти скины в файловой системе

## 5. Проверка работы

После настройки:
- Зайдите на сервер - должно прийти Embed сообщение о входе
- Получите достижение - должно прийти Embed сообщение
- Умрите - должно прийти Embed сообщение о смерти

Все сообщения будут с аватарами игроков!

## Решение проблем

### Бот не подключается
- Проверьте правильность токена
- Убедитесь, что бот добавлен на сервер Discord
- Проверьте логи сервера

### Сообщения не отправляются
- Проверьте правильность `channelId`
- Убедитесь, что бот имеет права на отправку сообщений в канал
- Проверьте, что `enabled: true` в конфиге

### Аватары не отображаются
- Если используется SkinRestorer, проверьте настройки
- Мод автоматически использует fallback на Crafatar API
- Проверьте интернет-соединение сервера
